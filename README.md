## 概要
このリポジトリには、今まで制作したゲーム作品を格納しています。  
以下のフォルダに、各作品データが格納されています。

## フォルダ構成
- `01_3D作品/`  
　→ 3Dゲームの「STICK×GANG!」を格納しています。

- `02_2D作品/`  
　→ 2Dゲームの「遺灰のなかでめざめて」を格納しています。

## 各フォルダ内について
各フォルダ内には、以下が含まれています。

- ビルド環境フォルダ
　→ ソースコードを格納しています。
- 実行環境フォルダ
　→ 実行ファイル（.exe）や作品紹介動画、作品概要書説明書などを格納しています。

## 動作環境
- Windows10 / Windows11  
- DirectX 11対応
